- Firebase account
- MongoDB database

### Installation
1. Clone the repository
```bash
git clone [repository-url]
cd SportZone-Claind
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
# Create .env file with Firebase config
VITE_FIREBASE_API_KEY=your_api_key

```

4. Start development server
```bash
npm run dev
```
